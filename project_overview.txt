Проект: Приложение для доставки еды

Обзор:
- Фронтенд на React
- Бэкенд на NestJS
- Используется Redis для кэширования

Фронтенд:
- Построен с использованием Create React App
- Основные скрипты: npm start, npm test, npm run build, npm run eject

Бэкенд:
- Построен с использованием NestJS
- Основные скрипты: npm run start, npm run start:dev, npm run start:prod

Настройка окружения:
- Установка зависимостей с помощью npm install в директориях client и server

Дополнительные заметки:
- Проект использует TypeScript как для фронтенда, так и для бэкенда
