Бэкенд (Server):
- Построен с использованием NestJS
- Основные скрипты:
  - npm run start: Запуск сервера в режиме разработки
  - npm run start:dev: Запуск сервера в режиме наблюдения
  - npm run start:prod: Запуск сервера в продакшн режиме
- Тестирование:
  - npm run test: Запуск юнит-тестов
  - npm run test:e2e: Запуск end-to-end тестов
  - npm run test:cov: Проверка покрытия тестами
- Использует TypeScript для типизации
